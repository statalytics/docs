## Description

Premium gifts can be purchased from <a href="https://statalytics.net/premium" target="_blank">our store page</a>. On the checkout page, you can choose to gift a premium plan to someone else.

Once a gift has been purchased, you will redirected to the gifts tab on <a href="https://statalytics.net/dashboard" target="_blank">our dashboard</a>, where you can copy the gift link. You can send this link to a friend for them to redeem, or simply redeem it yourself.

For more information on premium plans, see [Premium Plans](/premium/plans/).
